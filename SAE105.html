Traitement des données

📅 Date : Janvier 2025
⏱️ Durée : 17 Heures
👥 Équipe : Projet Collectif

Réalisation d’un programme Python pour automatiser la collecte, le traitement et l’export de statistiques à partir de données issues d’une API en ligne (PokeAPI). Découverte de l’utilisation d’API REST, du format JSON et de l’export Markdown.

Objectifs

  • ▸ Récolter des données via une API en ligne (PokeAPI)
  • ▸ Mettre en forme les données selon les informations désirées
  • ▸ Effectuer des traitements statistiques sur les données récoltées
  • ▸ Exporter les statistiques dans un format Markdown

Travaux réalisés

  • ▸ Étude du fonctionnement des API REST et du format JSON
  • ▸ Utilisation de la bibliothèque requests en Python pour interroger la PokeAPI
  • ▸ Traitement et filtrage des données reçues (listes, dictionnaires)
  • ▸ Calcul de statistiques (ex : nombre de Pokémon par type, moyenne de points de vie, etc.)
  • ▸ Génération d’un rapport au format Markdown

Résultat

  • ▸ Programme fonctionnel permettant d’automatiser la collecte et l’analyse de données issues d’une API
  • ▸ Statistiques exportées dans un format lisible et partageable (Markdown)

Conclusion

Cette SAE m’a permis de découvrir l’exploitation d’API REST, la manipulation de données JSON en Python et l’automatisation de traitements statistiques. J’ai également appris à générer des rapports en Markdown, un format très utilisé dans le monde professionnel. Ce projet m’a donné une vision concrète de l’automatisation et de l’analyse de données à partir de sources externes.

Problèmes rencontrés Solutions apportées
Difficultés à comprendre la structure des données JSON de l’API Utilisation de la documentation officielle et tests avec des requêtes simples
Problèmes pour automatiser l’export Markdown Recherche d’exemples et adaptation de la génération de texte en Python

Si c'était à refaire : Je passerais plus de temps à explorer d’autres API et à approfondir la génération de rapports automatisés.
